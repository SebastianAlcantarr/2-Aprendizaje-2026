Esta semana más que un proyecto tenemos varias libretas en jupyter a forma de tutorial, las cuales hay que completar:

1. La libreta [`reg_lineal`](https://github.com/IA-UNISON/3-Aprendizaje/blob/main/reg_lineal.ipynb), si bien no sigue exactamente la notación usada en clase, si es lo suficientemente clara (a mi parecer) para repasar las nociones de aprendizaje automático.

2. La libreta [`reg_logistica`](https://github.com/IA-UNISON/3-Aprendizaje/blob/main/reg_logistica.ipynb) es un poco más complicada, pero sigue siendo relativamente fácil para reforzar lo visto en clase, aunque cambie algo de la notación.

3. La libreta [`regularizacion_logistica`](https://github.com/IA-UNISON/3-Aprendizaje/blob/main/regularizacion_logistica.ipynb) habla de una operación clave en todo proceso de generación de un modelo de predicción:  la regularización. Esto no lo vimos en clase, pero asumo que los estudiantes de la LCC están listos para asumir este tipo de retos.

Todas las libretas tienen el mismo valor (1/3 de la calificación). Si no está completa una libreta, cuenta su avance.
